# Useful Tools

- **AWS CLI & AWS Shell** — automate tasks and scripting.
- **SAM CLI / Serverless Framework** — package and deploy serverless stacks quickly.
- **AWS CDK / Terraform** — infrastructure as code at scale.
- **LocalStack** — local AWS API simulation for dev/test.
- **k9s / kubectl / eksctl** — manage EKS clusters.
- **pgAdmin/DBeaver** — manage RDS databases.
- **CloudWatch Logs Insights** — fast log querying; save common queries.
