# Certification Guide

Path
- Start: Cloud Practitioner (CLF-C02)
- Core: Solutions Architect Associate (SAA-C03)
- Next: Developer Associate (DVA) or SysOps (SOA)
- Advanced: Solutions Architect Professional, Security Specialty

Study pattern
- Read exam guide → take free practice questions → build small labs → do at least 2 timed practice exams.

Key topics (SAA)
- VPC design, IAM, S3 durability/cost classes, EC2/ECS/EKS/Lambda, RDS/Aurora/DynamoDB, caching, messaging (SQS/SNS/Kinesis), HA/DR, cost optimization.

Exam day
- Timebox: ~1.2 min/question; flag long ones. Re-read “MOST cost-effective / SIMPLEST” cues.
