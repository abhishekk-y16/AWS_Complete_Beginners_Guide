# Learning Paths

Beginner (2-4 weeks)
- Cloud Practitioner videos, hands-on with S3/EC2/Lambda, set budgets/alarms.

Builder (4-6 weeks)
- Deploy a serverless API (API GW + Lambda + DynamoDB), host SPA on S3+CloudFront, set CI/CD.

Container track (4-6 weeks)
- Docker/ECR basics → ECS/Fargate → EKS fundamentals → observability (CloudWatch/X-Ray).

Data track (4-6 weeks)
- S3 lake → Glue crawler/job → Athena queries → Redshift Serverless basics.
