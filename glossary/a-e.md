# Glossary A–E

- **Availability Zone (AZ)** — isolated data center within a region.
- **AWS CLI** — command-line interface to manage AWS services.
- **Auto Scaling Group (ASG)** — manages EC2 fleet size automatically.
- **Bucket** — top-level container for objects in S3.
- **CloudFormation** — infrastructure-as-code service for AWS.
- **CloudFront** — CDN for caching and delivering content.
- **EBS (Elastic Block Store)** — block storage volumes for EC2.
- **EC2 (Elastic Compute Cloud)** — virtual machines in AWS.
- **ECS (Elastic Container Service)** — AWS-native container orchestrator.
- **EFS (Elastic File System)** — managed NFS file system.
