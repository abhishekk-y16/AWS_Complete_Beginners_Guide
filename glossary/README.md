## ðŸŽ¯ Most Important Terms for Beginners

### Essential Services (Learn First)
- **[EC2](a-e.md)** - Virtual servers in the cloud
- **[S3](p-t.md)** - Object storage service
- **[IAM](f-j.md)** - User and permission management
- **[VPC](u-z.md)** - Private network on AWS
- **[Lambda](f-j.md)** - Serverless compute
- **[RDS](p-t.md)** - Managed relational databases

### Infrastructure Terms
- **[Region](p-t.md)** - Geographic location of AWS data centers
- **[Availability Zone (AZ)](a-e.md)** - Isolated data centers within a region
- **[Subnet](p-t.md)** - Subdivision of VPC network
- **[Security Group](p-t.md)** - Virtual firewall for instances

### Common Acronyms
- **ARN** - Amazon Resource Name (unique resource identifier)
- **CDN** - Content Delivery Network (CloudFront)
- **CIDR** - Classless Inter-Domain Routing (IP address ranges)
- **EBS** - Elastic Block Store (storage volumes)
- **SNS** - Simple Notification Service
- **SQS** - Simple Queue Service
