# Glossary U–Z

- **VPC (Virtual Private Cloud)** — isolated network in AWS.
- **VPC Endpoint** — private connection to AWS services without internet.
- **WAF (Web Application Firewall)** — filters HTTP(S) traffic for apps.
- **Well-Architected Framework** — AWS best practices pillars.
- **X-Ray** — distributed tracing service.
- **Zero Trust** — security approach assuming no implicit trust; verify every request.
