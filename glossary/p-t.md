# Glossary P–T

- **RDS (Relational Database Service)** — managed relational databases.
- **Region** — geographic area containing multiple AZs.
- **Route 53** — DNS and traffic routing service.
- **S3 (Simple Storage Service)** — object storage service.
- **Security Group (SG)** — stateful virtual firewall for ENIs.
- **SNS (Simple Notification Service)** — pub/sub messaging.
- **SQS (Simple Queue Service)** — managed message queue.
- **Step Functions** — serverless workflow orchestration.
- **Terraform** — IaC tool used with AWS.
