# Glossary F–J

- **Fargate** — serverless compute for ECS/EKS tasks/pods.
- **Glacier** — archival storage tiers within S3.
- **IAM (Identity and Access Management)** — controls authentication and authorization.
- **Internet Gateway (IGW)** — VPC component enabling internet access.
- **Kinesis** — streaming data platform (Data Streams/Firehose/Analytics).
- **KMS (Key Management Service)** — managed encryption key service.
