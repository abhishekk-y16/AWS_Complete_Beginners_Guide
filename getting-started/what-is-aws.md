## Simple Definition

**AWS = Rent computers and services from Amazon instead of buying your own**

Instead of:
- ‚ùå Buying physical servers ($10,000+)
- ‚ùå Maintaining data centers
- ‚ùå Hiring IT staff
- ‚ùå Upgrading hardware every 3 years

- ‚úÖ Rent servers by the hour ($0.01/hour)

## What Can You Do with AWS?

### 1. Host Websites and Apps
```
Your website on AWS:
- No servers to buy
- Handles millions of visitors
- Scales automatically
- Costs start at $5/month
```

**Example:** Netflix runs entirely on AWS

### 2. Store Files and Data
```
S3 (Simple Storage Service):
- Store photos, videos, backups
- $0.023 per GB/month
- Unlimited storage
- 99.999999999% durability
```

### 3. Run Databases
```
RDS (Relational Database Service):
- MySQL, PostgreSQL, etc.
- Automated backups
- No database admin needed
- Starts at $15/month
```

### 4. Process Data
```
Lambda (Serverless Computing):
- Run code without servers
- Pay per execution
- Auto-scales to millions
- First 1M requests free/month
```
